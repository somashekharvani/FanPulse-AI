# FanPulse AI: Changelog

All notable changes to the FanPulse AI Operating System will be documented in this file.

## [v1.0.0] - 2026-07-16
### Added
- **Stage 1 (Core OS)**: 3D Digital Twin, ticket analyzer, voice synthesizer, and basic chat companion.
- **Stage 2 (Human Experience)**: FanPulse Match Day Companion AI tabbed dashboard (Group Connect, live status tracking, emergency low-battery warnings, privacy-safe lost child matching, split routes, offline navigators, group statistics, and ASCII Mini Map).
- **Stage 3 (Global Command Center)**: 3D rotating earth globe covering 16 locations, crisis simulation triggers, war room updates, global score average cards, and Autopilot ending recap splash screens.
- **Security & Privacy**: HS256 JWT validations, dynamic CORS headers, organizer MFA checkpoints, and granular user consent profiles.
